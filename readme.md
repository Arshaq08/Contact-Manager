# 📞 Contact Manager - Python

This is a simple Python-based contact manager that allows you to:

- 🔍 Search for a contact by name  
- 📋 Display all stored contacts  
- 📦 Extract multiple contact numbers by name

---

## 💡 Features

1. **Search a Contact** – Find a phone number by contact name.
2. **Display All Contacts** – View all contacts stored in the system.
3. **Extract Multiple Contacts** – Get numbers of multiple contacts by entering their names.
